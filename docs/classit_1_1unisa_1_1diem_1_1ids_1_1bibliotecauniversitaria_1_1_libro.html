void it.unisa.diem.ids.bibliotecauniversitaria.Libro.decrementaCopieDisponibili ( )
inline

Decrementa il numero di copie disponibili di questo libro nel catalogo.

Questo metodo viene invocato tipicamente dopo la registrazione di un prestito, come postcondizione del caso d'uso "Registrazione Prestito".

Precondition
Il libro deve avere almeno una copia disponibile (verificato da haCopieDisponibili()).
Postcondition
Il conteggio delle copie disponibili del libro è diminuito di una unità.

◆ haCopieDisponibili()

boolean it.unisa.diem.ids.bibliotecauniversitaria.Libro.haCopieDisponibili ( )
inline

Verifica se è presente almeno una copia disponibile di questo libro per un nuovo prestito.

Questo metodo è utilizzato come condizione di validità (IF-6.4) durante la procedura di "Registrazione Prestito".

Returns
true se è disponibile almeno una copia del libro; false altrimenti.
Precondition
una copia del libro è già registrato nel sistema.
Postcondition
Lo stato di disponibilità del libro viene accertato.

void it.unisa.diem.ids.bibliotecauniversitaria.Libro.incrementaCopieDisponibili ( )
inline

Incrementa il numero di copie disponibili di questo libro nel catalogo.

Questo metodo viene invocato tipicamente dopo la registrazione di una restituzione, come postcondizione del caso d'uso "Registrazione Restituzione".

Precondition
Il libro deve essere stato precedentemente prestato, aggiunto e quindi restituito.
Postcondition
Il conteggio delle copie disponibili del libro è aumentato di una unità.
